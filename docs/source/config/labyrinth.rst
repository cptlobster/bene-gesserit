The Labyrinth
=============
Bene Gesserit uses `Iocaine <https://iocaine.madhouse-project.org>`_ to generate the Labyrinth, a prison for LLM
scrapers and other unsavory bots to consume.

See :ref:`How A Scraper Is Caught` for more information on how a bot can end up in the labyrinth.

Setting Up the Labyrinth
------------------------
Iocaine requires a set of "corpus" files to setup the Markov chain. Bene Gesserit has tools to automatically download
some files for you.

- **By URL**: Download a text file directly by URL.
- **Project Gutenberg**: Download a book from Project Gutenberg by its ID.
- **By Path**: Provide a path on the filesystem that Bene Gesserit will copy.

Additionally, it requires a "words" file for generating links to deeper documents. The words file consists of single
words separated by newlines.

Banishment
----------
You can enable options in the configuration to banish clients completely to the labyrinth after a certain amount of
violations (honeypot links, ratelimits, etc.). The amount of violations that can trigger this are configurable. Since
this is based on the cookie generated by Anubis, their access will reset once the cookie expires or the user clears
their site data.

Slow Mode
---------
Enabling slow mode will establish a request transfer rate limit, which reduces the speed at which response data is
served. This is mainly intended to waste scrapers' time. You should configure this in a way that keeps scrapers waiting,
but doesn't cause them to lose interest. Exact guidance on this will be added at a later date.
